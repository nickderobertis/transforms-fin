# transforms-fin

## Overview

A set of Transforms meant for financial analysis to be used with the datacode package

## Getting Started

Install `transforms_fin`:

```
pip install transforms_fin
```

A simple example:

```python
import transforms_fin

# Do something with transforms_fin
```
